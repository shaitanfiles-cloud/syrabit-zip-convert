## Setup & Secrets
Add `PINECONE_KEY`/`PINECONE_INDEX=syrabit-ahsec` env vars (Railway). Implement `ensure_pinecone_index()` mirroring `ensure_vector_index()`: Serverless AWS us-east-1, 1024-dim (Cohere multilingual-v3.0), cosine metric. [docs](https://docs.rs/crate/pinecone-sdk/latest)

```python
# In config.py or utils.py
from pinecone import Pinecone
def ensure_pinecone_index():
    pc = Pinecone(api_key=os.getenv('PINECONE_KEY'))
    if 'syrabit-ahsec' not in pc.list_indexes().names():
        pc.create_index(
            name='syrabit-ahsec',
            dimension=1024,
            metric='cosine',
            spec=ServerlessSpec(cloud='aws', region='us-east-1')
        )
```

## PineconeVectorRetriever
Create `retrievers/pinecone_vector.py`: Matches `MongoVectorRetriever` interface (query/upsert/delete).
- ID: `chunk_id`; metadata: `{'subject_id', 'chapter_id', 'board_id', 'embedding_model'}`.
- Filters: `pc_index.query(vector=emb, top_k=k, filter={'board_id': 'ahsec'})`. [docs.pinecone](https://docs.pinecone.io/integrations/langchain)

```python
class PineconeVectorRetriever:
    def __init__(self):
        self.pc = Pinecone(api_key=os.getenv('PINECONE_KEY'))
        self.index = self.pc.Index('syrabit-ahsec')
    
    def query(self, embedding, top_k=5, filter=None):
        res = self.index.query(vector=embedding, top_k=top_k, include_metadata=True, filter=filter)
        return [{'chunk_id': m['id'], 'vs_score': match.score} for match in res.matches]
    
    def upsert(self, vectors):  # [{'id': chunk_id, 'values': emb, 'metadata': {...}}]
        self.index.upsert(vectors=vectors, batch_size=100)
```
