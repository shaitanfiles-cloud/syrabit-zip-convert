## Critical Fix #2: SPA → Prerender (30min)

**Solution**: **Vite-plugin-ssr** or **Prerender.io** (your programmatic SEO pages).

### Option A: Vite Pre-render (Recommended)
```bash
npm i vite-plugin-ssr -D
```

**vite.config.js**:
```js
import { VitePluginSSR } from 'vite-plugin-ssr/plugin'

export default {
  plugins: [
    // Your existing plugins
    VitePluginSSR({
      prerender: true,  // Pre-render SEO routes
      routes: ['./src/pages/**/*.page.js']  // Your topic pages
    })
  ]
}
```

**src/pages/ahsec-physics-waves.page.js**:
```js
export default function Page() {
  return (
    <>
      <SEO title="AHSEC Physics Waves Notes | Class 12" />

  )
}
```
