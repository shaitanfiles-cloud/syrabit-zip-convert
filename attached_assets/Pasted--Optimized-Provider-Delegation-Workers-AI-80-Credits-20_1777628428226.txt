## llm.py Logic Update (Python)
```python
PROVIDERS = {
    'chat': ['workers_ai', 'openrouter', 'groq'],  # 80/10/10
    'embed': ['workers_ai', 'cohere'],
    # ...
}
async def select_provider(feature, input_text, lang='en'):
    if is_assamese(lang): return 'sarvam_ai'
    load = get_gateway_metrics(feature)
    if load < 0.8: return 'workers_ai'
    return PRIORITY[feature][int(load * 10) // 2]  # Credits fallback
```
Gateway unifies billing/caching; monitor for 80% Workers cap. Total tier-1 credits: ~$5k entry (scale to $400k). [developers.cloudflare](https://developers.cloudflare.com/ai-gateway/usage/providers/)
