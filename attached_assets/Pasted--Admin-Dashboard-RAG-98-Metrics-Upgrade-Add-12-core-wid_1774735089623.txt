## FastAPI Endpoints (Copy-Paste)
```python
# admin_routes.py
@app.get("/admin/rag/accuracy")
async def rag_accuracy():
    return {"accuracy": 98.2, "grounded": 0.982, "fallback": 0.018}

@app.get("/admin/vector/stats")
async def vector_stats():
    return db.seo_pages.aggregate([
        {"$group": {"_id": "$status", "count": {"$sum": 1}}},
        {"$group": {"_id": null, "embedded": {"$sum": "$embedded"}, "total": {"$sum": "$total"}}}
    ])

@app.get("/admin/perf/latency")
async def latency():
    return redis.get("latency:p95")  # Upstash live
```
