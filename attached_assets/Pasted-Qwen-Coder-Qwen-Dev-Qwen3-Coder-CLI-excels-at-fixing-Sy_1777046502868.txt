## Setup for Replit
1. Install via pip in Replit Shell: `pip install qwen-code-cli` (uses qwen3-coder model via API/Ollama). [alibabacloud](https://www.alibabacloud.com/blog/boost-your-coding-workflow-with-qwen-code-a-practical-guide_602991)
2. Authenticate: `qwen-code login` with Alibaba API key (free tier: 1M tokens/day).
3. Run in your Syrabit repo: `qwen-code --task "Fix [issue]"`.
