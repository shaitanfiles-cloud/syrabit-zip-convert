## Frontend: Single Blue Source Card
```jsx
// SourcesCard.jsx - Replace chips list
function SourcesCard({ sources }) {
  return (
    <div className="source-card">
      {/* Watermark Header */}
      <div className="source-watermark">
        <span className="syrabit-icon">🤖</span>
        <span>Syrabit.ai AHSEC Browser</span>
      </div>
      
      {/* Unified Pages List */}
      <div className="source-pages">
        {sources.pages.map((page, i) => (
          <a key={i} href={page.url} className="source-link">
            📖 {page.name}
          </a>
        ))}
      </div>
      
      {/* Footer Stats */}
      <div className="source-stats">
        {sources.total_chunks} chunks • {sources.coverage}
      </div>
    </div>
  )
}
```
