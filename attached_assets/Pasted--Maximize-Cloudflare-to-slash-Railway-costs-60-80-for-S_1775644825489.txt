### 2. **Edge Workers for Light Logic** (20% savings)
```javascript
// /api/rate-limit.js (Cloudflare Workers - Free tier)
export default {
  async fetch(req, env, ctx) {
    const userId = req.cf.userId;
    const limit = await env.KV.get(`rate:${userId}`);
    if (limit > 30) return new Response("Rate limited", {status: 429});
    // Proxy to Railway only if under limit
    return fetch(req.url.replace("light-api.", "syrabit."), req);
  }
}
```
**Handles**: Auth checks, rate limiting, CORS, static responses
