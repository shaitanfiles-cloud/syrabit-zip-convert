Syrabit.ai CMS Doc Editor upgrades for WordPress/Gutenberg parity and syllabus-blog pipeline:

**Content Tab**
- **Embed Syllabus**: "Insert Subject Syllabus" dropdown → auto-pulls resolved scope MD from `/api/syllabus/resolve/{board}/{class}/{stream}/{subject}` → inserts as `# Subject Syllabus\n\n[content]` block
- **Syllabus Templates**: Add "Syllabus Intro" button → inserts structured template (`## Syllabus Overview\n### Key Topics\n### Chapters\n### Exam Guidelines`)
- **Chapter Links**: Auto-detect chapter shortcodes → converts to internal links (`[Chapter 1: Kinematics](/ahsec/class-11/physics/kinematics)`)
- **Live WordPress Preview**: Split-pane shows exact `/learn/[slug]` rendering (iframe via React iframe-resizer)

**SEO & Meta Tab**
- **Syllabus Signals**: Auto-populate `primary_keyword` from syllabus Key Topics; add `syllabus_board/class` structured fields for GEO
- **Enhanced Preview**: Google SERP + Perplexity snippet simulators (AI-generated citation preview)
- **Canonical Auto-Link**: If `linked_subject_id` set, auto-generates `<link rel="canonical" href="/[board]/[class]/[subject]">`

**GEO Tags Tab**
- **Syllabus Authority**: Auto-extracts "AHSEC 2026 Syllabus", "Class 11 Physics Board Exam" from scope → boosts citation likelihood

**Toolbar/List Enhancements**
- **Syllabus Quick-Actions**: Filter list by `linked_subject_id`; bulk "Link to Syllabus" checkbox
- **Versioning**: "Save as New Revision" → creates dated duplicate with diff view in MDXEditor
- **Workflow Integration**: "Hand Off to Content Editor" → seeds Content Editor chapters from post content

**Backend Ties**
- New `/api/cms/{id}/link-syllabus/{scope_id}`: Embeds syllabus stub + sets foreign keys
- Syllabus publish auto-creates CMS stub pre-filled with this editor's fields

**Impact**: Syllabus stub card → 1-click "Edit Content" → MDXEditor with embedded syllabus → chapters added → merged blog. Single-pane Gutenberg experience across entire pipeline. [mdxeditor](https://mdxeditor.dev)
