How this baseline was measured. Real Chrome Lighthouse could not run inside this environment because the container's NixOS profile is missing several of Chrome's runtime libraries (libudev/libgbm/freetype-harfbuzz ABI mismatch on the bundled binary). Instead, the numbers below are computed from the actual on-disk built artifacts: HTML and CSS file sizes were measured directly, and FCP / LCP estimates use Lighthouse's own "Simulated Throttling" network model (Slow 4G: 1.6 Mbps, RTT 150 ms, single connection per origin) applied to the measured critical request chain. A real Chrome Lighthouse run on the deployed Cloudflare Pages URL is tracked in follow-up #865.
